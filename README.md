# Number row characters:

- \! - exclamation mark
- \@ - at symbol  
- \# - hashtag  
- \$ - dollar sign
- \% - percent  
- \^ - caret
- \& - ampersand
- \* - star
- \( - left bracket
- \) - right bracket
